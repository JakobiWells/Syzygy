# Syzygy

Guides for getting your life in order — plus an interactive eclipse planner.

Two things live here:

- **Guides** (`/`, `/guides/:slug`) — practical markdown guides with embedded interactive components (e.g. the periodic table of spices).
- **Eclipse Planner** (`/eclipse`) — a Mapbox globe for planning solar/lunar eclipse viewing: eclipse paths from the NASA 5MCSE catalog, location scoring by historical cloud cover, day/night terminator, ISS passes, meteor showers, planetary transits, light pollution, weather overlays, and hotel search.

## Setup

```bash
npm install
cp .env.example .env   # then fill in your keys
```

| Env var | Purpose |
| --- | --- |
| `VITE_MAPBOX_TOKEN` | Mapbox GL access token (required for the eclipse map) |
| `VITE_WEATHER_PROXY_URL` | Cloudflare Worker proxying OpenWeatherMap tiles (`workers/weather-proxy`) |
| `VITE_HOTELS_PROXY_URL` | Cloudflare Worker proxying Google Places (`workers/hotels-proxy`) |

## Local development

```bash
npm run dev
```

Opens at `http://localhost:5173`.

## Adding a guide

1. Create a new `.md` file in `src/guides/`
2. Add frontmatter at the top:

```markdown
---
title: Your Guide Title
slug: your-guide-slug
description: One-line description shown on the home page.
tags: [tag1, tag2]
---

## First Section

Your content here.
```

The guide appears automatically on the home page at `/guides/your-guide-slug`.
Headings at `##` and `###` level are picked up for the table of contents.
Interactive React components can be embedded with `<div data-component="name"></div>`
(registered in `src/pages/Guide.jsx`).

## Data pipeline

- `public/eclipseCatalog.json` is generated by `src/eclipse/generate_catalog.py` from
  NASA Besselian elements (`besselian.csv`) + the 5MCSE catalog (`5MCSEcatalog.txt`).
- `src/eclipse/issArchive.json` (historical ISS TLEs) is generated by `scripts/fetchIssArchive.mjs`.
- `npm run horizon` — terrain horizon precomputation (`scripts/computeHorizon.mjs`).
- `npm run bortle` — light-pollution raster processing (`scripts/processBortle.py`, needs Python 3).

## Workers

Cloudflare Workers live in `workers/`; each has its own `wrangler.toml`:

- `weather-proxy` — OpenWeatherMap tile proxy (secret: `OWM_KEY`), edge-cached.
- `hotels-proxy` — Google Places hotel search cached in D1 (secrets: `GOOGLE_PLACES_KEY`,
  optional `CJ_BOOKING_AID` / `CJ_AIRBNB_AID`). Schema in `schema.sql`.

## Build & deploy

```bash
npm run build     # output in dist/
npm run deploy    # vite build + wrangler pages deploy
```

Deploy anywhere static. `vercel.json` is pre-configured for Vercel (SPA rewrite + cache headers).
